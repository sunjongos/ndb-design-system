# Patient Portal UI Kit — NDB

Patient-facing dashboard after sign-in. Appointments, records, messages.

**Components**
- `Sidebar.jsx` — left rail navigation
- `TopBar.jsx` — search + notifications + user
- `AppointmentCard.jsx` — upcoming appointment hero
- `QuickActions.jsx` — 4-up action tiles
- `RecentVisits.jsx` — list of recent visits
- `VitalStats.jsx` — summary stats

Open `index.html` to see the stitched dashboard.
