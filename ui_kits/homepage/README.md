# Homepage UI Kit — NDB

Marketing homepage for Namyangju Baek Hospital. Patient-facing surfaces for finding a department, booking an appointment, and reading about the hospital.

**Components**
- `Header.jsx` — sticky GNB with brand mark, nav, primary CTA
- `Hero.jsx` — full-width hero with headline + appointment CTA
- `DepartmentGrid.jsx` — 4-up grid of specialty centers
- `DoctorRail.jsx` — horizontally-scrollable doctor cards
- `InfoStrip.jsx` — hours / directions / contact band
- `Footer.jsx` — institutional footer with columns

Open `index.html` to see the full stitched homepage.
